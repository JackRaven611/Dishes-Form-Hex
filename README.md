# Dishes Form
Simple form with variable fields and API comunication, done in around 10 hours.

## Tech Stack

node: 16.17.0 <br>
npm: 8.15.0 <br>
react: 18.2.0 <br>
typescript: 4.9.5 <br>
styed-components: 5.3.10 <br>



## Run Locally

Install dependencies

```bash
  npm install
```

Start dev version

```bash
  npm start
```

Start build

```bash
    npm run build
```
